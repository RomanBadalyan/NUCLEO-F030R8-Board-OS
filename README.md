# Описание

Данный проект представляет собой __полное воссоздание оригинальной отладочной платы STM32 Nucleo-F030R8__ на базе микроконтроллера __STM32F030R8T6.__ Плата разработана с соблюдением всех габаритных размеров и расположения разъемов оригинального устройства, обеспечивая полную совместимость с экосистемой Arduino и ST Morpho.

![рисунок самой платы](image_link_here)

## :dart: Ключевые особенности

* __100% совместимость__ с оригинальной Nucleo-F030R8 по габаритам (70x82.5 мм) и распиновке
* __Двойная функциональность:__ работает как отладочная плата и как программатор для других STM32
* __Полный набор интерфейсов:__ Arduino Uno R3, ST Morpho, SWD, USB Virtual COM Port
* __Профессиональная 4-слойная структура РСВ__ (улучшенная версия оригинальной 2-слойной)
* __Готовность к производству:__ все файлы для заказа РСВ и компонентов включены

## :hammer_and_wrench: Технические характеристики

### Микроконтроллеры

* **Основной MCU:** STM32F030R8T6 (Cortex-M0, 64 КБ Flash, 8 КБ RAM, 48 МГц)
* **Программатор/отладчик:** STM32F103CBT6 (Cortex-M3, встроенный ST-LINK/V2-1)

### Питание

* **Вход USB:** 5V (500 мА максимум)
* **Вход VIN**: 7-12V (до 800 мА)
* **Выходы: 3.3V** (500 мА), 5V (до 800 мА)
* **Управление питанием:** Интеллектуальное включение MCU части

### Разъемы

* **Arduino Uno R3:** XP5, XP6, XP8, XP9 (полная совместимость с шилдами)
* **ST Morpho:** XP7, XP10 (полный доступ ко всем GPIO)
* **Отладка:** XP4 (SWD для внешних плат)
* **USB:** Mini-B для программирования и питания

### Особенности

* **Кварцы:** 8 МГц (в ST-LINK), 32.768 кГц (для RTC)
* **Индикация:** 3 светодиода (PWR, COM, USER)
* **Кнопки:** USER и RESET
* **Измерение потребления:** Перемычка XP6 для амперметра
